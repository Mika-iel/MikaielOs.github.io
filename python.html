Cafe Sales Data Cleaning

Cafe Python Zip Files

This project focused on cleaning and preparing a raw cafe sales dataset using Python and Pandas. The objective was to transform an inconsistent transactional dataset into a clean, analysis-ready dataset suitable for future analysis and visualisation.

The dataset was sourced from Kaggle and contained missing values, unknown entries, inconsistent formatting, and incorrect data types that required cleaning and transformation before analysis.

The data cleaning process included:

  • Exploring dataset structure, data types, and missing values using Pandas
  • Standardising column names and categorical values
  • Cleaning inconsistent item information and invalid entries
  • Handling missing values based on column context
  • Removing unnecessary columns with excessive unknown values
  • Converting numerical and date columns into appropriate formats

Additional validation and reconstruction steps were performed, including:

  • Validating product pricing information
  • Reconstructing missing values where possible
  • Calculating total spending using quantity and unit price
  • Checking the accuracy of the cleaned dataset

The final cleaned dataset was exported as an analysis-ready file. This project demonstrated the use of Python for data cleaning, transformation, validation, and preparing raw data for further analytics.

Countries of the World Web Scraping

Web Scraping Countries Zip Files

This project focused on extracting structured data from a webpage using Python web scraping techniques. The objective was to collect information from an HTML page, transform the extracted data into a structured format, and create a dataset suitable for further analysis.

The webpage contained information about countries around the world, including country names, capital cities, population, and land area. Python libraries were used to request, parse, and extract the required information from the webpage.

The web scraping process included:

  • Sending requests to the webpage using the Requests library
  • Parsing HTML content using BeautifulSoup
  • Inspecting HTML structures to identify relevant elements and patterns
  • Extracting country information from specific HTML classes
  • Storing extracted information into Python lists

The extracted data included:

  • Country names
  • Capital cities
  • Population values
  • Area measurements (km²)

The scraped information was converted into a Pandas DataFrame and exported as a CSV file, transforming webpage data into a structured dataset ready for future analysis.

Future improvements include cleaning numerical columns, performing additional data analysis, and creating visualisations from the collected dataset.
